How do I create an automation that turns on the kitchen lights when motion is detected, but only after sunset?
10:23 AM
I'll help you create that automation! Here's how to set it up:

1. Trigger: Motion sensor state change to "on"
2. Condition: Sun is below horizon
3. Action: Turn on kitchen lights

Here's the YAML configuration:

automation:
  - alias: "Kitchen lights on motion"
    trigger:
      - platform: state
        entity_id: binary_sensor.kitchen_motion
        to: "on"
    condition:
      - condition: sun
        after: sunset
    action:
      - service: light.turn_on
        target:
          entity_id: light.kitchen
